# SocialApp

Pet project.

Main technologies and skills to practice:
- Docker
- GraphQL
- React
- TypeScript
- Rails 6.0+

## Run

```bash
$ docker-compose up --build
```

## Interacting

```bash
# Initial setup - creates admin, sets up & seeds database
$ ./docker/setup.sh
# Rails console
$ ./docker/run-console.sh
# Alpine sh
$ ./docker/run-shell.rh
# PG command line
$ ./docker/run-postgres.sh
```
